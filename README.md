## 功能性需求與非功能性需求
  
# 功能性需求
  1. 室內定位技術。
  2. Beacon定位位置。
  3. 標籤系統。
  4. 資料自動上傳。

    1. 反應性: 提供監控人員隨時查詢資料
    2. 使用性: 標籤系統連結建築物室內架構
